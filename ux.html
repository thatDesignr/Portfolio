Project: User Experience

UX task

Hike One asked us to redesign the user flow for one of the three biggest Dutch health insurers. We started off with researching competitors, doing user tests and then redesigned the UI and UX of ZEKUR.

Researching competitors

We first researched all 3 health insurances and wrote down our findings. We wrote down good and bad points.

User testing

We then did 20+ user tests on these 3 websites. We chose to improve the website with the worst userflow, which was ZEKUR.

User flow of ZEKUR

To show the client what we are going to improve, using the user tests, we visualized the current userflow.

Heuristic Evaluation

To further research the things that need to be improved on the ZEKUR website, we used the Heuristic Evaluation from the Nielsen Norman Group.

Final redesigned website

After a month of hard work we finished the prototype of the website.

Presenting to the client

In the end we could present our work at the Hike One office in Amsterdam. We made a professional presentation and learned alot from presenting to Hike One.
